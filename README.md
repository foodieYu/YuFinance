# 魚的質感客製化記帳本

專為行動端與桌面端優化的個人記帳 Web 應用程式。採用現代化的前端技術棧與無伺服器後端架構，提供流暢的記帳體驗與直觀的財務報表。

## 🛠️ 技術棧 (Tech Stack)
- **Frontend Framework**: React 18 + Vite
- **Styling**: Tailwind CSS / shadcn/ui
- **Database & Auth**: Supabase
- **Charts**: Recharts
- **State Management**: React Hooks & Context API

## ✨ 核心功能 (Features)
- 📱 **響應式設計 (RWD)**：支援手機端底部導覽列與桌面端側邊欄。
- 📊 **動態數據報表**：自動計算歷史總資產、各帳戶真實餘額與消費圓餅圖。
- ⚡ **極速記帳體驗**：為手機觸控優化的大按鈕與動態分類連動表單。
- 🔒 **企業級安全**：導入 Supabase RLS (Row Level Security) 保護個人帳務隱私。
